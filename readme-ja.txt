=== yaWeb拍手 ===Author: ksksAuthor URI: http://trial-run.net/Version: 0.1=== Description ===WordPress 上で動く web拍手です。WordPress のデータベースで動作、ダッシュボードから解析結果を閲覧することが出来ます。スクリプト配布版のweb拍手を使い続けているWordPressユーザさんとかどうでしょ？=== Installation ===1. web拍手フォームの設定	WordPress の「固定ページ」から web拍手フォームを作成します。		「拍手をもっと送る」ボタンおよび、コメントフォームは	ショートコード [webclap] で作成されます。		また、付属の webclap.php を使用しているテーマのディレクトリに移動、	固定ページのテンプレートに Web Clap を使うと、表示がされっぽくなります	2. web拍手ボタンの設定	サイドバーの「ウィジェット」から、	「利用出来るウィジェット」の中から「web拍手」を選択。		・ウィジェットのタイトル		・ボタンのリンク先（＝web拍手フォームのurl）		・ボタンの画像（url、テキストリンクの場合空白）		・リンクのテキスト（若しくは、imgタグのalt属性）3. 解析画面について	サイドバーの「ダッシュボード」＞「yaWeb拍手」		・今日の拍手履歴		・昨日の拍手履歴（昨日 〜 2週間前まで）		・今月の拍手履歴（今月 〜 1年前まで）		・拍手コメント ログ4. web拍手フォームのカスタマイズ	フォームについては WordPressの「固定ページ」で生成されていますが、	デフォルトの WordPress のテンプレートを使いたくない場合もあると思います。		一手間かかりますが、	同梱されている webclap.php を現在使用しているテーマのディレクトリにコピー、	固定ページのテンプレートを webclap にすることで、	シンプルなフォーム画面にすることが出来ます。	=== Upgrade Notice ==== v0.2 =・多言語化に対応= v0.1.1 =・解析画面が表示されないバグを修正= v0.1 =・リリース
